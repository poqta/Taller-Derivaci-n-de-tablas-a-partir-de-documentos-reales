Taller: Derivación de tablas a partir de documentos reales

Este repositorio contiene el desarrollo del taller de derivación de datos a partir de un documento real, utilizando una factura como documento base.

Contenido del repositorio

- Archivo de Enterprise Architect que contiene los modelos.
- Script SQL con la creación de la base de datos, tablas, inserción de datos y consulta final.
- Documento del taller con la explicación del proceso.

El objetivo del taller es analizar una factura real y derivar, a partir de ella, un modelo de base de datos. Para esto se identificaron entidades, atributos y relaciones, las cuales fueron representadas en Enterprise Architect y luego implementadas en SQL Server.
